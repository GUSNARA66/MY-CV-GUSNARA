I use GitHub as a place to document my learning progress and showcase my projects.

Understanding Tailwind

Tailwind CSS helped me speed up my development process by using utility classes. Instead of writing custom CSS from scratch, I can apply styling directly in HTML

This method makes my code cleaner, more consistent, and easier to maintain, especially as a beginner
